## ✨ Overview

**TrackDown** is a minimalist, browser-based productivity app built around the [Pomodoro Technique](https://en.wikipedia.org/wiki/Pomodoro_Technique). It gives students and focused workers a distraction-free way to time their sessions, log their work, and visualize daily progress — all without needing an account, a server, or a subscription.

> Built with vanilla HTML, CSS, and JavaScript. No dependencies. No frameworks. Just focus.


## 🔥 Features

| Feature | Description |
|---|---|
| ⏱️ **Pomodoro Timer** | Customizable work and break intervals with Start / Pause / Stop controls |
| 🏷️ **Session Labeling** | Tag each session with a subject or topic for organized tracking |
| 📊 **Progress Meter** | Visual thermostat-style canvas showing hours worked vs. daily goal |
| 📅 **Time Views** | Toggle between Today, This Week, and This Month summaries |
| 🔥 **Streak Counter** | Day-streak tracking to keep your momentum going |
| 📝 **Session History** | Auto-logged list of completed sessions with timestamps |
| ✏️ **Manual Entry** | Add sessions retroactively (for when you forget to start the timer!) |
| 🔔 **Audio Chime** | Gentle notification sound when a session ends |
| 🪟 **Glassmorphism UI** | Clean, modern glass-panel design using the *Outfit* font |

## 🚀 Getting Started

No installs, no build steps. Just clone and open.

### Prerequisites

- Any modern web browser (Chrome, Firefox, Edge, Safari)

### Installation

```bash
# 1. Clone the repository
git clone https://github.com/Kris-Zar/TrackDown.git

# 2. Navigate into the project folder
cd TrackDown

# 3. Open in your browser
open index.html
# or just double-click index.html in your file explorer
```

That's it. TrackDown runs entirely in the browser — no server required.

## 🧠 How It Works

1. **Set your session** — enter a subject label and customize your work/break durations (default: 25 min work / 5 min break).
2. **Hit Start** — the countdown begins. Stay focused until the chime sounds.
3. **Review your progress** — completed sessions are automatically logged in the history panel and reflected in the progress meter.
4. **Track your streak** — come back daily to keep your streak alive and build a lasting study habit.

<br/>

## ⚙️ Customization

You can adjust defaults directly in `script.js` or via the UI:

- **Work duration** — set any value in minutes
- **Break duration** — set any value in minutes
- **Daily goal** — set your target study hours (default: 2h)
